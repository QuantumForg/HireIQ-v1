# Assets

Place the brand logo here as **`logo.png`**:

```
assets/logo.png
```

The dashboard automatically loads `assets/logo.png` for the sidebar brand and
the browser tab icon. If the file is absent, it falls back to the built-in
"HQ" gradient mark. Supported formats: PNG, JPG, SVG, WEBP, GIF (PNG with a
transparent background is recommended).
