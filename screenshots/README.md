# Screenshots

The README references the images below. Save each screenshot here using the
**exact filename** (PNG recommended) so they render in `README.md` and on GitHub.

| Filename | What to capture |
|:--|:--|
| `pipeline_execution.png` | Your terminal after running `python -m src.pipeline` (showing stages `[1/4]`–`[4/4]` and `PIPELINE SUCCESSFULLY COMPLETED`). |
| `dashboard_overview.png` | Dashboard **Overview** page — hero banner + the six metric cards + pipeline timeline. |
| `dashboard_analytics.png` | Dashboard **Analytics** page — score distribution / charts. |
| `dashboard_leaderboard.png` | Dashboard **Leaderboard** page — the ranked candidate table. |
| `dashboard_validation.png` | Dashboard **Validation** page — the green "Submission Valid" card. |

## Tips

- Capture at a wide browser width (≈1280px) for crisp dashboard shots.
- Use Windows Snipping Tool (`Win + Shift + S`) and save as PNG.
- Keep file sizes reasonable (under ~1 MB each) for fast README loading.
